# 04 Interaction Capability
## Purpose
Collect evidence for accessibility, task completion, and error-resistant interactions.
Use this folder for audits and usability execution traces.
## Expected artifacts
- Accessibility exports in 11y/
- Task execution runs in 	ask-runs/
- Error-protection evidence in error-protection/
## Naming convention
Use YYYY-MM-DD_<artifact>.<ext>, for example 2026-03-10_a11y-scan.json.
## Redaction reminder
Remove user-identifiable data and sensitive form values from interaction records.
