# 03 Compatibility
## Purpose
Collect evidence that supported browsers and devices render and behave correctly.
Use this folder for compatibility matrices and device captures.
## Expected artifacts
- Browser support matrix in rowser-matrix/
- Device screenshots in eal-device-screenshots/
- Optional notes for known compatibility limitations
## Naming convention
Use YYYY-MM-DD_<artifact>.<ext>, for example 2026-03-10_ios-safari-dashboard.png.
## Redaction reminder
Mask user data, location details, and private account information in captures.
