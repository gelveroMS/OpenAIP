# 01 Functional Suitability
## Purpose
Collect evidence that core features work as intended for required user journeys.
Use this folder for functional test outputs and supporting screenshots.
## Expected artifacts
- Playwright reports in playwright-report/
- Workflow screenshots in screenshots/
- Optional issue notes linked to failing steps
## Naming convention
Use YYYY-MM-DD_<artifact>.<ext>, for example 2026-03-10_login-flow.png.
## Redaction reminder
Remove or blur personal data, tokens, and secret values before saving artifacts.
